dbx-key
^^^^^^^
Dropbox Application key.

:Default: ``None``

dbx-secret
^^^^^^^^^^
Dropbox Application secret.

:Default: ``None``

dbx-token
^^^^^^^^^
Application access token.

:Default: ``None``

dbx-path
^^^^^^^^
File path in Dropbox account.

:Default: ``/output``

dbx-archive
^^^^^^^^^^^
Archive path in Dropbox account

:Default: ``/archive``

dbx-autorename
^^^^^^^^^^^^^^
Rename file if it already exists.

:Default: ``True``
